Project: Image Colorization using Deep Learning

📝 About the Project

This project allows users to upload a black and white image and receive a colorized version generated using a deep learning model. It combines machine learning and web development to provide an interactive experience for end users.

🛠️ Technologies Used

  • Frontend: HTML, CSS, JavaScript (Flask templates)
  • Backend: Python with Flask
  • Model: Siggraph2017 pretrained image colorization model
  • Image Processing: C++ app using deep learning binaries in Codeblocks

⚙️ How It Works

  1. User uploads a black & white image.
  2. Image is saved and passed to a C++ application colorize_app.
  3. The app uses a deep learning model to colorize the image.
  4. Result is displayed and can be downloaded.

🚀 Future Enhancements

  • Support for bulk uploads
  • Drag-and-drop interface
  • Real-time progress indicator
  • Option to choose different colorization models
